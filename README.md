# AI Chief of Staff

This repository contains a production-minded scaffold for an AI-powered Personal Life OS. It includes a Next.js dashboard, a Node.js API, negotiation logic, and supporting docs.

## What's Included
- **Frontend**: Next.js dashboard in `frontend/`
- **Backend**: Express API in `backend/`
- **Docs**: Architecture, API contracts, scenarios, and setup in `docs/`

## Quick Start
```bash
cd backend
npm install
npm run dev
```

```bash
cd frontend
npm install
npm run dev
```
